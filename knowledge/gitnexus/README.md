# GitNexus Code Asset Store

This folder stores reusable industrial automation code assets generated or curated by the learning loop.

## Rules

- Put SCL examples under `scl/`.
- Put WinCC Classic VBS examples under `wincc-classic-vbs/`.
- Put WinCC Unified JavaScript examples under `wincc-unified-js/`.
- Put CLI bridge examples under `cli-control/`.
- Every asset must include source assumptions and tag/keyword names in comments.
- Commit generated assets with meaningful messages when Git is initialized.
