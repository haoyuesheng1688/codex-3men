# SCL Industrial Rules Baseline

Updated: 2026-05-31 17:50:44 +00:00
Keywords: SCL_CASE_OF, SCL_OPTIMIZED_ACCESS

## Digest

SCL examples generated by this project must use clear block interfaces, avoid absolute DB access in optimized blocks, and prefer CASE ... OF state machines for sequenced industrial control. Loop constructs must include bounded execution logic so PLC scan time and watchdog behavior remain predictable.
